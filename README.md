## Kovan Mod – Card Art Roller Guide

This repository contains multiple systems, but this document is specifically meant to help anyone who is only interested in the Card Art Roller system.

If you're here just for that feature, you can ignore the rest of the codebase and follow the guide below.

### If your goal is to understand or modify only the Card Art Roller system, focus on the following flow:

Navigate until you get to KovanModCode, then search for the Directories.

#### Go to:

Abstract/KovanModCardModel.cs

This is where card art data is requested and processed.

#### Go to:

Utility/VanillaArtPool.cs

This is where the actual rolling logic happens.
